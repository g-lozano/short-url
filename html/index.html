URL Shortener

User Stories

1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.

2. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.

3. When I visit that shortened URL, it will redirect me to my original link.

How to Use

    Generate a new short url by adding the following at the end the URL:

    /new/<URL>

    An example result would be:

    https://lil-url-gl.herokuapp.com/new/http://www.google.com

    URLs must have the format http://www.example.com

Example Results

    https://lil-url-gl.herokuapp.com/notashortURL

    {"error":"Invalid short URL."}

    https://lil-url-gl.herokuapp.com/new/ or https://lil-url-gl.herokuapp.com/new/invalid_format.com

    {"error":"URL format is invalid."}

    https://lil-url-gl.herokuapp.com/new/http://www.urldoesnotexist.com/

    {"error":"Website does not exist."}

    https://lil-url-gl.herokuapp.com/new/http://www.google.com

    {"original_url":"http://www.google.com","short_url":"https://lil-url-gl.herokuapp.com/7GLJ"}
